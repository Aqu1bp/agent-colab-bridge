# Code of Conduct

This project aims to be a practical, respectful place to collaborate.

Use public issues and pull requests for technical work. Do not post credentials,
private Worker URLs, account identifiers, notebook output containing secrets, or
exploit details in public threads.

Unacceptable behavior includes harassment, personal attacks, disclosure of private
information, or intentionally disruptive conduct. Maintainers may moderate issues,
pull requests, and discussions to keep the project focused and safe.

For security concerns, follow `SECURITY.md`.
